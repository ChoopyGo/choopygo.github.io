## Features

- **Coming Soon Landing Page** - Beautiful hero section with gradient design
- **Interactive Terminal** - Command-line interface with games and features
  - Type `help` to see available commands
  - Type `about` to learn about ChoopyGO and timeline
  - Type `game` to play mini-games
  - Type `contact` for contact information
  - Type `clear` to clear the terminal
- **Development Timeline** - Interactive roadmap showing progress from Q4 2025 to Q3 2026
  - Bilingual support (Thai & English)
  - Scroll animations with staggered reveals
  - Color-coded milestones
  - Responsive design
- **Navigation Menu** - Sticky header with smooth scrolling
  - Links to Home and Timeline sections
  - Scroll effects with frosted glass backdrop
- **Responsive Design** - Optimized for mobile, tablet, and desktop
